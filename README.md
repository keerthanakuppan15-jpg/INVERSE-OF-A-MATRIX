# INVERSE-OF-A-MATRIX
## Aim:
To write a python program to find the inverse of a matrix
## Equipment’s required:
1. 	Hardware – PCs
2. 	Anaconda – Python 3.7 Installation / Moodle-Code Runner
## Algorithm:
Step 1:

Start the program and import the NumPy library to perform matrix operations.

Step 2:

Define the given square matrix using a 2-dimensional array (list of lists).

Step 3:

Check whether the matrix is invertible and compute its inverse using the np.linalg.inv() function.

Step 4:

Display the inverse matrix as the output.
## Program: 
```
#Program to find the inverse of a matrix.
#Developed by:keerthana k
#RegisterNumber:212225230137
import numpy as np
print(np.linalg.inv([[1,0,3],[-1,2,-2],[2,3,-1]]))
```
